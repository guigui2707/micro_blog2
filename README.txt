UTILISATION DU SITE MICRO_BLOG


J'ai réalisé toutes les fonctionnalités demandés.

=========BASE DE DONNEES ===============

Nom: micro_blog
identifiant: root
mdp: (aucun mdp)

La base de donnees est incluse dans le code sous le nom de micro_blog.sql


==========Les utilisateurs ============

compte ADMINISTRATEUR :

nom: Administrateur
email: admin@admin.com
mdp : admin

Le compte administrateur a TOUS LES PRIVILIEGES (modification et suppression de tous les 
messages).

compte MEMBRES : 

nom: membre1
email: membre1@membre1.com
mdp: membre1

nom: guillaume
email: guillaume@guillaume.com
mdp: guillaume

D'autres membres peuvent etre inscrits.


=========Mon Compte===================

Dans cette rubrique possibilité de SE CONNECTER ou de CREER UN COMPTE

Si lorsque l'on veut se connecter on n'est pas rediriger vers la page d'accueil c'est
qu'il y a une erreur de saisie des login

Lors de la création d'un compte des messages d'alertes s'afficheront si les champs ne sont 
pas correctement remplis ou que l'utilisateur existe déjà
Si la création du compte s'effectue avec succès, un message d'alerte vous avertira, et vous
pourrez vous connecter au blog


========= L'outil de recherche ========

L'outil de recherche est composé d'un champ de recherche où l'on peut rechercher un mot contenu
dans un corps de message. 
Pour valider la recherche, il suffit de taper sur entrée.
